# Introduction

This documentaion goes through how a PSP can integrate with Intergiro Acquring. This includes how to make and an Authorization, Verification, Capture, Refund and Cancelation.

There is also documentation for how to make rules for merchants in the acquirer. These rules can automatically reject authorizations, captures, refunds or cancelation based on boolean statements.
